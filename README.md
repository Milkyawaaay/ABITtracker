# ABITtracker — трекер поступления и доставки документов

**ABITtracker** — это веб-приложение для отслеживания процесса поступления в ВУЗы и организации грузоперевозок документов. Система помогает абитуриентам контролировать дедлайны, требования к документам и доставку оригиналов в приемные комиссии.

---

## Структура проекта

Проект состоит из 4 основных страниц:

- **Авторизация** (регистрация абитуриента)
- **Вход** (аутентификация пользователя)
- **Главная страница** (трекер поступления и перевозки)
- **Профиль абитуриента** (личные данные и история заявок)

---

## Функционал проекта 

### Главная страница

На главной странице абитуриент может:

- Отслеживать статусы подачи документов в выбранные ВУЗы
- Получить результат обработки:
  - При подтверждении / отказе отображается список ВУЗов
- Получить напоминания о дедлайнах и требованиях

### Страница профиля абитуриента

На странице профиля отображается:

- **ФИО, контакты и выбранные ВУЗы**
- **Текущий статус поступления** (баллы ЕГЭ, этапы рассмотрения)
